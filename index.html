📱 Guide d'utilisation
🔧 Installation sur ton téléphone
›
Sur iPhone (Safari) :
1. Ouvre Safari → va sur djrecap.github.io/djrecap
2. Appuie sur l'icône Partager (carré avec flèche)
3. "Sur l'écran d'accueil" → Ajouter

Sur Android (Chrome) :
1. Ouvre Chrome → va sur djrecap.github.io/djrecap
2. Chrome propose automatiquement "Ajouter à l'écran d'accueil"
3. Ou menu ⋮ → "Ajouter à l'écran d'accueil"
➕ Ajouter une soirée
›
1. Appuie sur le bouton + en bas à droite
2. Remplis le type, lieu, date et ville
3. Pour un mariage : renseigne les prénoms et email des mariés
4. Note la soirée (1 à 5 étoiles) et l'ambiance piste (1 à 10)
5. Ajoute les sons qui ont cartonné
6. Sélectionne les styles joués
7. Ajoute tes notes sur la salle
8. Appuie sur Enregistrer
✉️ Envoyer un mail aux mariés
›
1. Ouvre une soirée de type Mariage
2. Appuie sur "Envoyer un mail de remerciement"
3. Vérifie l'email et les prénoms des mariés
4. Ajoute un message personnel si tu veux
5. Appuie sur Envoyer

⚠️ EmailJS doit être configuré dans ton profil pour que l'envoi fonctionne.
📊 Stats & Export PDF
›
Stats avancées (Premium) :
Accède à tes stats via l'onglet Stats en bas — soirées par mois, répartition par type, top sons, top styles.

Export PDF (Premium) :
Depuis l'onglet Stats → appuie sur "Exporter mon bilan PDF" → un fichier HTML est généré → ouvre-le dans ton navigateur → Fichier → Imprimer → Enregistrer en PDF.
📧 Configuration EmailJS
① Créer un compte EmailJS
›
1. Va sur emailjs.com
2. Clique sur "Sign Up" → crée un compte avec ton Gmail
3. Confirme ton adresse email

💡 Le plan gratuit inclut 200 emails/mois — largement suffisant pour commencer.
② Connecter ton Gmail
›
1. Dans EmailJS → "Email Services" → "Add New Service"
2. Choisis Gmail
3. Clique "Connect Account" → connecte ton Google
4. Donne-lui un nom ex: "Mon Gmail DJ"
5. Clique "Create Service"

💡 Note le Service ID — il ressemble à : service_abc123
③ Créer un template email
›
1. EmailJS → "Email Templates" → "Create New Template"
2. To Email : {{to_email}}
3. Subject : Merci pour votre confiance — {{from_name}}
4. Body : {{message}}
5. Clique "Save"

💡 Note le Template ID — il ressemble à : template_xyz789
④ Configurer dans DJ Recap
›
1. Ton nom → "Account" → copie ta Public Key
2. Dans DJ Recap → Profil → Configuration EmailJS
3. Remplis les 3 champs :
  • Public Key : user_XXXX
  • Service ID : service_XXXX
  • Template ID : template_XXXX
4. Clique Sauvegarder
❓ Questions fréquentes
Je change de téléphone, je perds mes données ?
›
Non ! Toutes tes données sont dans le cloud. Reconnecte-toi avec ton email sur le nouveau téléphone et tout est là. ✅
D'autres DJs peuvent voir mes soirées ?
›
Non, impossible. Chaque DJ a un compte totalement isolé. Tes données sont strictement privées et sécurisées. 🔒
L'app fonctionne sans internet ?
›
La consultation des soirées déjà chargées fonctionne en offline. Pour ajouter une soirée ou envoyer un mail, une connexion internet est nécessaire.
